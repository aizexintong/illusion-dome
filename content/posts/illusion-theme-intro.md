幻梦（Illusion）是一款为追求极致视觉体验而生的 Hugo 博客主题。它不仅仅是一个主题，更是一场关于数字美学的探索。

## 设计理念

幻梦主题的设计灵感来源于自然界中的梦幻景象——晨雾中的紫藤花、夜空中的繁星、林间的萤火虫。我们试图将这些自然之美转化为数字世界的视觉语言。

### 梦幻色彩

主题采用紫色为主色调，搭配粉色和绿色作为辅助色，营造出柔和、梦幻的视觉氛围。浅色模式下呈现「朦胧幻梦」的淡雅气质，深色模式下则展现「夜色幻梦」的深邃神秘。

```css
:root {
  --c-primary-400: #786C8A;
  --c-secondary-400: #856876;
  --c-accent-400: #4D7A60;
}
```

### 磨砂玻璃美学

卡片、侧边栏、导航等 UI 元素统一采用磨砂玻璃效果，带来通透、轻盈的视觉感受。

```css
.card {
  background: var(--c-surface-glass);
  backdrop-filter: blur(16px);
}
```

## 核心特性

- 双主题切换（浅色/深色）
- Canvas 粒子特效（白天糖果彩带 / 夜间星光萤火虫）
- 本地搜索
- 响应式设计
- SEO 优化
- 多语言支持
- 完整的数据驱动页面

幻梦主题让每一篇博文都像是一件艺术品，阅读变成了一种享受。
