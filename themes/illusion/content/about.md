# 关于我的故事

欢迎来到我的个人空间！我是 **爱则心痛**，一名热爱技术的开发者，专注于Web开发、移动应用和创意编程。在这里，我分享我的技术探索、生活感悟和项目经验。

## 技术栈与专长

我专注于现代Web技术栈，包括 **Vue.js 3**、**React 18**、**Rust** 和 **TypeScript**，同时也涉足移动端开发（Flutter）和桌面应用开发。我的技术理念是：**代码不仅是工具，更是创造的艺术**。

### 技术理念

- **用户体验至上**：我相信优秀的产品始于卓越的用户体验，每个细节都值得精心打磨
- **代码即艺术**：编写优雅、可维护的代码是我的追求，技术应该服务于创造而非限制
- **持续学习**：技术世界日新月异，保持好奇心和学习热情是不断进步的动力
- **开源精神**：分享知识、回馈社区，让技术惠及更多人
- **创新思维**：在解决问题时寻求创新方案，突破传统思维局限

## 技术领域

### 前端开发

- **现代框架**：Vue.js 3、React 18、Qwik、SolidJS
- **构建工具**：TypeScript、Vite、Webpack、Rollup
- **UI/UX设计**：关注用户体验和界面设计，追求美学与功能的平衡
- **移动端**：Flutter、React Native、Tauri

### 后端开发

- **系统编程**：Rust、Node.js、Python、Go
- **API设计**：RESTful API、GraphQL、微服务架构
- **数据库**：PostgreSQL、MySQL、MongoDB、Redis
- **DevOps**：Docker、Kubernetes、CI/CD、Serverless

### 移动开发

- **跨平台**：Flutter、Dart、React Native
- **原生开发**：Android、Kotlin、iOS、Swift
- **性能优化**：关注应用性能、内存管理和用户体验

### 开发工具

- **版本控制**：Git、GitHub Actions、GitLab CI
- **代码质量**：ESLint、Prettier、Husky、Commitlint
- **项目管理**：敏捷开发、团队协作、文档编写

## 项目经验

在多年的开发经历中，我参与并主导了多个项目，涵盖：

1. **企业级应用**：为大型企业开发内部管理系统和客户服务平台
2. **开源项目**：维护多个开源项目，贡献代码和文档
3. **个人项目**：探索新技术、实验创新想法
4. **社区贡献**：参与技术社区，分享知识和经验

## 写作与分享
